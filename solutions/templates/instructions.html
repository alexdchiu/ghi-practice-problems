GHI Problems

The abbreviation GHI stands for Graphical Human Interface. This is a newer abbreviation than UI which stands for User Interface. We use the more specific GHI abbreviation because humans are the ones that look at and interact with HTML. There are other types of interfaces for machines, and those are APIs, or application programming interfaces.

This packet of problems allows you to practice and learn HTML tags, Django templates, and contexts passed to those templates.

All tests for these problems run in the browser. You do not have to run python manage.py test to see the results.

The files that you will change are in the problems/ Django app. You will be asked to modify:

  • Zero or more HTML files in the problems/templates/ directory
  • Zero or more Python files in the problems/views/ directory
  • Zero or more CSS files in the problems/static/ directory

You will likely find the Built-in template tags and filters documentation a nice reference for these exercises.
